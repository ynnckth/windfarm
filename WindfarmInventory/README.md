# Windfarm Inventory

**Service providing inventory details of all managed windfarms** 

Install: 
> `$ npm install`

Run: 
> `$ npm start`
> => https://localhost:5002

Docker build:
> `$ docker build -t windfarm-inventory .`

Docker run:
> `$ docker run -p 5002:5002 windfarm-inventory`
> => http://localhost:5002
